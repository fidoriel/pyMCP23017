# pyMCP23017
 Python Raspberry Pi MCP23017 with PullUp-support

 Pinout used in this library

   8   |--U--| 7
   9   |     | 6
   10  |     | 5
   11  |     | 4
   12  |     | 3
   13  |     | 2
   14  |     | 1
   15  |     | 0
   V   |     | NC
   GND |     | NC
   NC  |     | RST
   SCL |     | A2
   SDA |     | A1
   NC  |-----| A0

The usage is similar to the common python PRi.GPIO libary, documented in the example.
